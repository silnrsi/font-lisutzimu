FONTLOG for the LisuTzimu fonts

This file provides detailed information on the LisuTzimu Font Software. This information should be distributed along with the LisuTzimu fonts and any derivative works.

Basic Font Information

LisuTzimu is a Unicode typeface family that supports all languages that use the Lisu script and its variants, and could be expanded to support other scripts.
